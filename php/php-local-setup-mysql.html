Setup Walkthrough

PHP + MySQL Local Development Setup

This step-by-step page turns your checklist into a full local workflow: project creation, PHP server, MySQL setup, connection file, and database read test.

Before You Start

  • These install commands are for Ubuntu or Debian-based Linux.
  • Run commands from the VS Code terminal unless noted otherwise.
  • If a command asks for sudo, enter your OS password.

Step 1: Open your project

Create a folder, enter it, and open it in VS Code.

mkdir php-project
cd php-project
code .

Create a file named index.php in this folder.

Step 2: Check PHP is installed

This prints your current PHP version.

php -v

Expected output looks like PHP 8.x.x.

If PHP is missing, install it:

sudo apt update
sudo apt install php

Step 3: Start the PHP server

This starts a local development server in your current folder.

php -S localhost:8000

Open http://localhost:8000 in a browser. Stop the server with Ctrl + C.

Step 4: Install MySQL

Install the MySQL server package.

sudo apt update
sudo apt install mysql-server

Check installation:

mysql --version

Step 5: Start MySQL

Check service status, start it if needed, and enable auto-start on boot.

sudo systemctl status mysql
sudo systemctl start mysql
sudo systemctl enable mysql

Step 6: Log into MySQL

Open the MySQL shell as an admin user.

sudo mysql

You should now see a mysql> prompt.

Step 7: Create your database and table

Run these SQL commands inside the MySQL prompt.

CREATE DATABASE learningphp;
SHOW DATABASES;
USE learningphp;

CREATE TABLE users(
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(255)
);

INSERT INTO users(name, email)
VALUES
('Kevin', 'kevin@gmail.com'),
('Bob', 'bob@gmail.com');

SELECT * FROM users;
EXIT;

Step 8: Install PHP MySQL extension

PHP needs MySQL drivers before PDO can connect.

sudo apt install php-mysql
php -m

Verify PDO, pdo_mysql, and mysqli appear in the module list.

Step 9: Create database.php connection file

Create database.php in your project root:

<?php
$host = '127.0.0.1';
$db   = 'learningphp';
$user = 'root';
$pass = '';
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";

$options = [
    PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::ATTR_EMULATE_PREPARES => false,
];

try {
    $pdo = new PDO($dsn, $user, $pass, $options);
} catch (PDOException $e) {
    die('Connection failed: ' . $e->getMessage());
}
?>

Step 10: Connect and read data in index.php

Replace your index.php with:

<?php
require_once 'database.php';

$sql = "SELECT * FROM users";
$result = $pdo->query($sql);
$users = $result->fetchAll();

echo '<pre>';
print_r($users);
echo '</pre>';
?>

Start the server again:

php -S localhost:8000

Open http://localhost:8000 and you should see user rows printed as an array.

Copy-Paste Library

Common PHP Snippets

Use these small building blocks when you need a quick example for variables, loops, strings, arrays, and everyday utility code. Each block is short enough to copy into a project and adapt.

Variables

Basic variable and output pattern

Use this when you want a clear starting point for a page title or display name.

<?php
$pageTitle = "Dashboard";
$userName = "Amina";

echo "<h1>" . htmlspecialchars($pageTitle) . "</h1>";
echo "<p>Welcome, " . htmlspecialchars($userName) . "</p>";
?>
Conditionals

Simple role check

Use this for admin vs user display logic or feature flags.

<?php
$role = $_SESSION["role"] ?? "guest";

if ($role === "admin") {
    echo "<strong>Admin panel</strong>";
} else {
    echo "User area";
}
?>
Loops

Render a list with foreach

This is the most common pattern for arrays and database result sets.

<?php
$links = ["Home", "About", "Contact"];

echo "<ul>";
foreach ($links as $link) {
    echo "<li>" . htmlspecialchars($link) . "</li>";
}
echo "</ul>";
?>
Arrays

Associative array access

Useful for form data, configuration, and API payloads.

<?php
$user = [
    "name" => "Amina",
    "email" => "amina@example.com",
    "active" => true,
];

echo $user["name"] . " - " . $user["email"];
?>
Functions

Reusable formatting helper

Keep helpers small and specific so they can be reused across pages.

<?php
function formatCurrency(float $amount): string
{
    return "$" . number_format($amount, 2);
}

echo formatCurrency(49.99);
?>
Includes

Shared layout pieces

Use include for optional pieces and require for files your page cannot work without.

<?php
require __DIR__ . "/partials/header.php";
include __DIR__ . "/partials/flash-message.php";
?>
Sanitizing

Safe HTML output

Escape values before rendering anything that came from a user or database.

<?php
function safeOutput(string $value): string
{
    return htmlspecialchars($value, ENT_QUOTES, "UTF-8");
}

echo safeOutput($comment);
?>

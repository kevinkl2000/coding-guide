Databases With MySQL and PDO

PDO is the standard, flexible way to connect PHP to MySQL. It supports prepared statements, exceptions, and multiple database drivers.

Creating a Connection

<?php
$dsn = "mysql:host=localhost;dbname=school_app;charset=utf8mb4";
$username = "root";
$password = "";

try {
    $pdo = new PDO($dsn, $username, $password, [
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
        PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    ]);
} catch (PDOException $exception) {
    exit("Database connection failed: " . $exception->getMessage());
}
?>

Prepared Statements

Prepared statements separate the SQL structure from user input, which prevents SQL injection when used correctly.

<?php
$stmt = $pdo->prepare("SELECT * FROM users WHERE email = :email");
$stmt->execute(["email" => $email]);
$user = $stmt->fetch();
?>

Insert Data Safely

<?php
$sql = "INSERT INTO users (name, email, created_at) VALUES (:name, :email, NOW())";
$stmt = $pdo->prepare($sql);
$stmt->execute([
    "name" => $name,
    "email" => $email,
]);
?>

Fetching Multiple Rows

<?php
$stmt = $pdo->query("SELECT id, title, status FROM tasks ORDER BY created_at DESC");
$tasks = $stmt->fetchAll();

foreach ($tasks as $task) {
    echo "<p>" . htmlspecialchars($task["title"]) . "</p>";
}
?>

Suggested File Structure

project/
|-- config/
|   |-- database.php
|-- public/
|   |-- index.php
|-- src/
|   |-- UserRepository.php
|-- templates/
|   |-- header.php

Keeping connection logic in one place makes changes easier and avoids duplicated credentials.

PDO Rules

  • Enable exception mode so failures are visible.
  • Use prepared statements for any query involving user data.
  • Store credentials outside public web folders.
  • Escape output when rendering database content into HTML.

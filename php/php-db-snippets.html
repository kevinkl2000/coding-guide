Database Data Snippets

Use these generic snippets when you need to work with data returned from a database. They focus on common tasks like fetching rows, handling empty results, writing records, and printing values safely.

Fetch

Read one row

A compact pattern for pulling a single record with a prepared statement.

<?php
$stmt = $pdo->prepare("SELECT * FROM table_name WHERE id = :id");
$stmt->execute(["id" => $id]);
$row = $stmt->fetch();

if ($row) {
    echo $row["name"];
}
?>
Fetch

Loop through rows

The standard pattern for listing many rows from a table.

<?php
$stmt = $pdo->query("SELECT id, name FROM table_name ORDER BY id DESC");
$rows = $stmt->fetchAll();

foreach ($rows as $row) {
    echo "<p>" . htmlspecialchars($row["name"]) . "</p>";
}
?>
Empty State

Handle no results

Show a clear message when a query returns nothing.

<?php
if (empty($rows)) {
    echo "No records found.";
} else {
    foreach ($rows as $row) {
        echo htmlspecialchars($row["name"]);
    }
}
?>
Insert

Save a new row

A generic insert statement using placeholders.

<?php
$stmt = $pdo->prepare("INSERT INTO table_name (name, status) VALUES (:name, :status)");
$stmt->execute([
    "name" => $name,
    "status" => $status,
]);
?>
Update

Update an existing row

Use this for editing a row after the user submits a form.

<?php
$stmt = $pdo->prepare("UPDATE table_name SET name = :name WHERE id = :id");
$stmt->execute([
    "name" => $name,
    "id" => $id,
]);
?>
Delete

Delete by id

A minimal delete pattern with a bound id value.

<?php
$stmt = $pdo->prepare("DELETE FROM table_name WHERE id = :id");
$stmt->execute(["id" => $id]);
?>
Pagination

Basic limit and offset

Useful for breaking large result sets into pages.

<?php
$limit = 10;
$offset = ($page - 1) * $limit;

$stmt = $pdo->prepare("SELECT * FROM table_name ORDER BY id DESC LIMIT :limit OFFSET :offset");
$stmt->bindValue(":limit", $limit, PDO::PARAM_INT);
$stmt->bindValue(":offset", $offset, PDO::PARAM_INT);
$stmt->execute();
?>
Output

Print database values safely

Always escape values before placing them into HTML.

<?php
foreach ($rows as $row) {
    echo "<h2>" . htmlspecialchars($row["title"]) . "</h2>";
    echo "<p>" . htmlspecialchars($row["body"]) . "</p>";
}
?>

Lesson 1

PHP Basics

PHP is a server-side language that can generate HTML, process form data, talk to databases, and return JSON. Files normally end with .php and are executed by the server before the browser sees the response.

How PHP Works

When a user requests a PHP page, the web server passes the file to the PHP interpreter. PHP runs the code on the server and sends the result back to the browser.

<?php
$language = "PHP";
echo "<h1>Welcome to $language</h1>";
?>

Output

Welcome to PHP

The browser does not see the PHP code. It only sees the generated HTML.

Variables and Data Types

PHP variables start with a dollar sign. Common types include strings, integers, floats, booleans, arrays, and null.

<?php
$name = "Amina";
$age = 21;
$height = 5.7;
$isStudent = true;
$skills = ["PHP", "HTML", "SQL"];

var_dump($name, $age, $height, $isStudent, $skills);
?>

Output

string(5) "Amina"
int(21)
float(5.7)
bool(true)
array(3) {
  [0]=> "PHP"
  [1]=> "HTML"
  [2]=> "SQL"
}

Strings and Output

Use echo to print output. Double quotes allow variable interpolation, while single quotes treat text more literally.

<?php
$user = "Kevin";
echo "Hello, $user";
echo '<br>Welcome to the guide';
?>

Output

Hello, Kevin

Welcome to the guide

  • Use a dot to concatenate strings.
  • Use htmlspecialchars() when printing user-provided values into HTML.

Conditionals and Loops

PHP supports if statements, switch, while, for, and foreach. Use foreach for arrays because it is clean and readable.

<?php
$topics = ["Syntax", "Forms", "Sessions", "MySQL"];

foreach ($topics as $topic) {
    echo "<li>" . htmlspecialchars($topic) . "</li>";
}

$score = 82;
if ($score >= 80) {
    echo "Pass";
} else {
    echo "Review again";
}
?>

Output

  • Syntax
  • Forms
  • Sessions
  • MySQL

Pass

Functions

Functions help you reuse logic and keep files organized. Add type hints when possible for clearer code.

<?php
function formatPrice(float $amount): string
{
    return "$" . number_format($amount, 2);
}

echo formatPrice(49.99);
?>

Output

$49.99

Arrays

Indexed arrays store ordered lists. Associative arrays store key-value pairs and are very common in web apps.

<?php
$user = [
    "name" => "Amina",
    "role" => "admin",
    "active" => true,
];

echo $user["name"];
?>

Output

Amina

Including Other Files

Split shared layout pieces into separate files using include or require. Use require when the file is essential.

<?php
require "partials/header.php";
include "partials/sidebar.php";
?>

Key Takeaways

  • PHP runs on the server and returns HTML or JSON.
  • Variables start with $ and arrays are central to everyday PHP work.
  • Use functions and include files to keep code maintainable.
  • Escape output with htmlspecialchars() when rendering user data.

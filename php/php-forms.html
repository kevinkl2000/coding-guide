Lesson 2

Forms and User Input

Forms are one of the most common PHP use cases. PHP reads submitted values through superglobals such as $_GET and $_POST.

GET vs POST

Method Best Use Notes
GET Search, filters, sharable URLs Values appear in the URL.
POST Logins, registrations, updates Values are sent in the request body.

Reading POST Data

<?php
if ($_SERVER["REQUEST_METHOD"] === "POST") {
    $name = trim($_POST["name"] ?? "");
    $email = trim($_POST["email"] ?? "");

    if ($name === "" || $email === "") {
        echo "Please complete all fields.";
    } else {
        echo "Thanks, " . htmlspecialchars($name);
    }
}
?>

Output

If fields are empty:

Please complete all fields.

If the form is valid:

Thanks, Amina

Validation Pattern

Validation should happen on the server even if JavaScript also checks fields in the browser.

<?php
$errors = [];

if ($name === "") {
    $errors[] = "Name is required.";
}

if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
    $errors[] = "Email address is not valid.";
}

if (empty($errors)) {
    echo "Form is valid.";
}
?>

Output

Form is valid.

Sticky Form Example

A sticky form keeps the user's typed values after validation fails.

<form method="post" action="">
    <input type="text" name="name" value="<?= htmlspecialchars($name ?? "") ?>">
    <input type="email" name="email" value="<?= htmlspecialchars($email ?? "") ?>">
    <button type="submit">Send</button>
</form>

Output

Amina
amina@example.com
Send

File Upload Basics

Use enctype="multipart/form-data" and inspect $_FILES. Always validate file type and size.

<?php
if (isset($_FILES["resume"]) && $_FILES["resume"]["error"] === UPLOAD_ERR_OK) {
    $tmpPath = $_FILES["resume"]["tmp_name"];
    $originalName = basename($_FILES["resume"]["name"]);
    move_uploaded_file($tmpPath, __DIR__ . "/uploads/" . $originalName);
}
?>

Best Practice

Never trust raw user input. Validate structure, sanitize output, and use prepared statements before data reaches a database query.

Form Checklist

  • Check the request method before reading form values.
  • Use null coalescing like $_POST["field"] ?? "".
  • Escape data when showing it back in HTML.
  • Validate on the server even if JavaScript exists.

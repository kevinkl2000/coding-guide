Model

Simple model with validation

Models represent your domain or database-backed records.

<?php
class Post
{
    public function __construct(
        public readonly string $title,
        public readonly string $body
    ) {
    }

    public function isValid(): bool
    {
        return trim($this->title) !== "" && trim($this->body) !== "";
    }
}
?>
Repository

Data access class

Keep SQL out of controllers and views by centralizing queries here.

<?php
class PostRepository
{
    public function __construct(private PDO $pdo)
    {
    }

    public function all(): array
    {
        return $this->pdo->query("SELECT * FROM posts ORDER BY created_at DESC")->fetchAll();
    }
}
?>
Router

Minimal front controller router

A front controller gives you one entry point for all requests.

<?php
$path = parse_url($_SERVER["REQUEST_URI"], PHP_URL_PATH);

switch ($path) {
    case "/":
        require __DIR__ . "/../controllers/HomeController.php";
        break;
    case "/posts":
        require __DIR__ . "/../controllers/PostController.php";
        break;
    default:
        http_response_code(404);
        echo "Page not found";
}
?>

Loop through data in a view

Views should stay simple: loop, escape, and display.

<?php foreach ($posts as $post): ?>
    <article>
        <h2><?= htmlspecialchars($post["title"]) ?></h2>
        <p><?= htmlspecialchars($post["body"]) ?></p>
    </article>
<?php endforeach; ?>

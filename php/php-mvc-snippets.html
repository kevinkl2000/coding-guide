Generic MVC Snippets for PHP Apps

These are broad building blocks for PHP apps that use an MVC-style structure. The snippets stay neutral so you can adapt them to any project.

Controller

Empty controller action

A controller method that is ready for you to fill in.

<?php
class Controller
{
    public function index(): void
    {
        $data = [];

        require __DIR__ . "/../views/index.php";
    }
}
?>
Model

Simple model shell

A basic model structure with a placeholder property and method.

<?php
class Model
{
    protected array $data = [];

    public function all(): array
    {
        return $this->data;
    }
}
?>
Repository

Generic repository

Keeps data access separate from controllers and views.

<?php
class Repository
{
    public function __construct(protected PDO $pdo)
    {
    }

    public function all(): array
    {
        return [];
    }
}
?>
Router

Minimal router

One entry point for routing requests to the right controller.

<?php
$path = parse_url($_SERVER["REQUEST_URI"], PHP_URL_PATH);

switch ($path) {
    case "/":
        require __DIR__ . "/../controllers/HomeController.php";
        break;
    default:
        http_response_code(404);
        echo "Page not found";
}
?>
View

Generic loop in a view

A small loop for rendering any list of items.

<?php foreach ($items as $item): ?>
    <div>
        <?= htmlspecialchars($item) ?>
    </div>
<?php endforeach; ?>

Auth

Login guard

A generic session check for protected pages.

<?php
session_start();

if (empty($_SESSION["user_id"])) {
    header("Location: /login.php");
    exit;
}
?>

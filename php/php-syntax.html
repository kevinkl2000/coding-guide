Lesson 2

Variables, Types, and Syntax

This lesson slows down and makes PHP syntax predictable. Once you understand variables, data types, operators, and arrays, most beginner confusion disappears.

Variable Rules

PHP variables start with $. Use descriptive names because your future code will become much easier to read.

  • Use $firstName instead of $f.
  • Prefer one responsibility per variable.
  • Update values intentionally instead of reusing the same variable for unrelated data.

Scalar Types and Debugging

Strings, integers, floats, and booleans are the core scalar types. Use var_dump() when you are unsure what a value really contains.

<?php
$product = "Keyboard";
$price = 89;
$rating = 4.6;
$inStock = true;

var_dump($product, $price, $rating, $inStock);
?>

Output

string(8) "Keyboard"
int(89)
float(4.6)
bool(true)

Operators

Operators let you transform and compare values. Learn arithmetic, comparison, concatenation, and logical operators early.

<?php
$subtotal = 50;
$shipping = 10;
$total = $subtotal + $shipping;
$isFreeShipping = $total >= 60;

echo "Total: $" . $total;
echo "<br>Free shipping: " . ($isFreeShipping ? "Yes" : "No");
?>

Output

Total: $60

Free shipping: Yes

Strings and Interpolation

Double-quoted strings can inject variable values directly. Single-quoted strings treat content more literally. Use concatenation when building output from many pieces.

<?php
$name = "Amina";
$course = "PHP";

echo "$name is learning $course.";
echo '<br>' . $name . ' is practicing arrays.';
?>

Output

Amina is learning PHP.

Amina is practicing arrays.

Arrays

Arrays are everywhere in PHP. Form data, database rows, configuration, and JSON are often represented as arrays.

<?php
$user = [
    "name" => "Amina",
    "skills" => ["PHP", "MySQL", "HTML"],
    "active" => true,
];

echo $user["name"];
echo "<br>" . $user["skills"][1];
?>

Output

Amina

MySQL

Type Casting and Safe Assumptions

PHP will sometimes convert values automatically. That flexibility is convenient, but you should not rely on guesses. Cast values deliberately when input can arrive as strings.

<?php
$quantityFromForm = "3";
$price = 25;
$total = (int) $quantityFromForm * $price;

echo $total;
?>

Output

75

A beginner-friendly rule: if a value comes from a user, URL, or external system, inspect it first and cast it intentionally.

Lesson Goal

  • Recognize common PHP data types immediately.
  • Know when to use interpolation versus concatenation.
  • Read arrays confidently because they appear in nearly every PHP app.
  • Use var_dump() instead of guessing.

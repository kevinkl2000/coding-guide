Lesson 2

Colors and Typography

Type, spacing, and color create clarity and visual hierarchy.

Typography

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

h1 {
    font-size: 2.5rem;
    letter-spacing: -0.02em;
}

Colors

body {
    color: #222;
    background: #f8f8f8;
}

.button {
    background: #1e40af;
    color: #fff;
}

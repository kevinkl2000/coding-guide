Lesson 2

Init, Clone, and Setup

Start new repositories correctly and configure identity before making commits.

Set Your Identity

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

New vs Existing Project

# New local repo
git init

# Clone remote repo
git clone https://github.com/org/repo.git

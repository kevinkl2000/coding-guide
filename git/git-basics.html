Lesson 1

Git Basics

Git is a version control system that records snapshots of your project over time.

Daily Starter Flow

git status
git add .
git commit -m "Short clear message"
git log --oneline

Key Terms

  • Working tree: files you are editing.
  • Staging area: changes prepared for commit.
  • Commit: saved checkpoint in history.
  • Branch: independent line of work.

Lesson 4

Branching Fundamentals

Branches let you isolate work safely and avoid breaking stable code.

Common Branch Commands

git branch
git switch -c ticket/123-login-bug
git switch dev
git branch -d ticket/123-login-bug

Naming Tips

  • Use prefixes: feature, bugfix, hotfix, ticket.
  • Include ticket number when available.
  • Keep names short but descriptive.

Lesson 6

Remote Collaboration

Teams collaborate through remote repositories and pull requests.

Sync Commands

git remote -v
git fetch origin
git pull origin dev
git push origin ticket/123-login-bug

Pull Request Flow

  • Push your ticket branch to origin.
  • Open pull request into dev.
  • Respond to review comments.
  • Merge only after checks pass.

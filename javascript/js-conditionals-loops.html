Lesson 2

Conditionals and Loops

Use control flow to make code react to different situations.

If Statements

if (count > 0) {
    console.log("Has items");
} else {
    console.log("Empty");
}

Loop Example

for (let i = 0; i < 3; i++) {
    console.log(i);
}

Lesson 3

Arrays and Objects

Use arrays for ordered data and objects for named values and configuration.

Array Example

const items = ["one", "two", "three"];
items.push("four");
console.log(items.length);

Object Example

const user = {
    name: "Amina",
    role: "admin"
};
console.log(user.name);

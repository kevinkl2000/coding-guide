Lesson 9

Indexes and Performance

Indexes speed up lookups, but every index also has maintenance cost on writes.

Create an Index

CREATE INDEX idx_users_email
ON users (email);

Use EXPLAIN

EXPLAIN
SELECT *
FROM users
WHERE email = 'a@example.com';

Performance Checklist

  • Select only columns you need.
  • Filter early with indexed columns.
  • Avoid unnecessary wildcard searches.
  • Review query plans with EXPLAIN.

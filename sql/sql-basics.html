Lesson 1

SQL Basics

SQL queries data in relational databases. Start by understanding tables, rows, columns, and simple SELECT statements.

Core Concepts

  • Database: a container of tables.
  • Table: a structure with rows and columns.
  • Row: one record.
  • Column: one attribute.

First Query

SELECT *
FROM users;

Use SELECT to read data. Start with small tables while learning.

Select Specific Columns

SELECT id, name, email
FROM users;

Practice

Write one query with * and one query with named columns from the same table.

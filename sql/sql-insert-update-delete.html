Lesson 6

Insert, Update, Delete

These statements modify data. Learn them with care and always test on small data first.

INSERT

INSERT INTO users (name, email)
VALUES ('Amina', 'amina@example.com');

UPDATE

UPDATE users
SET email = 'new@example.com'
WHERE id = 10;

DELETE

DELETE FROM users
WHERE id = 10;

Never run UPDATE or DELETE without a WHERE clause unless you intend to affect all rows.

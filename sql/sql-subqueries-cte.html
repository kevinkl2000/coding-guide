Lesson 8

Subqueries and CTEs

Use subqueries and CTEs to break complex logic into manageable query blocks.

Subquery Example

SELECT *
FROM users
WHERE id IN (
    SELECT user_id
    FROM orders
    WHERE total > 500
);

CTE Example

WITH high_value_orders AS (
    SELECT user_id, total
    FROM orders
    WHERE total > 500
)
SELECT user_id, COUNT(*) AS order_count
FROM high_value_orders
GROUP BY user_id;

Lesson 3

Sorting and Limiting Results

Control result order and size for readable outputs and efficient views.

ORDER BY

SELECT id, name, created_at
FROM users
ORDER BY created_at DESC;

LIMIT and OFFSET

SELECT *
FROM users
ORDER BY id DESC
LIMIT 10 OFFSET 20;

Tip

Always pair LIMIT with ORDER BY for predictable results.

Lesson 3

Lists and Tables

Lists organize grouped items. Tables present structured data.

Lists

<ul>
    <li>Item one</li>
    <li>Item two</li>
</ul>

Table Structure

<table>
    <tr><th>Name</th><th>Age</th></tr>
    <tr><td>Amina</td><td>21</td></tr>
</table>
